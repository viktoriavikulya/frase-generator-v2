# Roadmap

## Completado
- Pipeline completo: render -> upload -> publish (IG + FB + Threads)
- Carruseles y singles
- Metricas semanales (performance_score, saves, reach)
- Selector de color desde el formulario
- Rotacion inteligente de fondos para evitar colores parecidos cerca
- Tokens permanentes (usuario del sistema Meta)
- Doctor local y auditoria opcional del Sheet
- Deteccion de frases duplicadas antes de registrar

## Proximo

- Transformador de inspiracion: convertir candidatos virales en versiones propias sin copiar literal
- Semana de puro carrusel para validar datos (forzar TIPO_INPUT=carousel en schedule)

## Ideas futuras
- Dashboard visual de metricas (top performers, tendencias por color)
- Seleccion inteligente de post basada en patrones historicos
- Alt text automatico para Instagram
- Location ID opcional para posts con identidad local
